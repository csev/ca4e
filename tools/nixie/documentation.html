Nixie Challenge

The Nixie Challenge gives learners a hands-on way to practice converting numbers between base-10 and base-2. Inspired by classic nixie tubes, the interface glows with each conversion to reinforce the connection between binary bits and the glowing decimal digit.

Accessibility Notes

  • Canvas elements include descriptive aria-label attributes.
  • Controls are accessible via keyboard (checkboxes support the space key to toggle).
  • Feedback messages use role="status" so screen readers announce the results.

Developer Notes

The nixie display is implemented in nixie.js using the Canvas 2D API. The glow effect is borrowed from the NixieDisplay component used in the Digital Gates tool.

Adding New Exercises

  1. Extend nixie.js with your exercise logic.
  2. Create new DOM sections in index.php for the activity UI.
  3. Reuse SimpleNixieDisplay to show decimal values.

Testing Checklist

  • Verify both exercises load with random values between 0 and 7.
  • Ensure check buttons evaluate answers correctly and handle invalid input.
  • Confirm the nixie display updates after a correct answer.
  • Try the easter egg shortcut (Ctrl + *) within the main tool to reveal the instructor-only features.
